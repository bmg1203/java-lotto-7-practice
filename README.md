# java-lotto-precourse

## 구현할 기능 목록


### 로또 구입
- 로또 구입 금액 입력
- 로또 구입 금액에 랜덤 번호 6개 생성
- 발행 로또 수량 및 번호 출력

### 로또 당첨 번호 발행
- 중복되지 않도록 6개 번호와 보너스 번호 입력
  - 로또 번호 범위 제약조건
  - 로또 번호 횟수 제약조건


### 로또 당첨 결과
- 당첨 조건과 당청금 관리
- 당첨 결과 통계 계산
  - 당첨 종류 별 당첨 횟수 계산 
  - 당첨금, 수익률 계산


### 입출력
- 입출력 기능 DTO로 통신
- 잘못된 입력을 받을 경우, IllegalArgumentException 발생 후, 재입력 요구
