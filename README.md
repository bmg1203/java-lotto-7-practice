### 풀이 방향

리팩토링을 통해서 정돈할 생각.

1. 로또 범위, 상금과 같은 상수는 enum 으로 모두 관리

   (enum 에서 _ 사용해도 되었나?)
2. 필요한 객체라 생각되는 것들

    > 로또(객체), 로또번호 생성기, 내가 가지고 있을 로또 용지들, 당첨 결과 계산 할 객체
3. 로또 머신에서 랜덤 볼들을 받아서 `MyLotto` 에 채워 넣음
4. 당첨 결과를 계산하고 그 정보를 저장
