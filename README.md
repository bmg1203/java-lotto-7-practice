# java-lotto-precourse

## 기능

### 1. `입력`
돈, 당첨 번호, 보너스 번호 입력받기
예외처리
1.잘못된 입력(숫자이외의 문자를 입력)
2.당첨 번호를 6개 입력하지 않을 때 
3.보너스 번호가 당첨 번호랑 같을 떄

### 2. `출력`
1.개수만큼 로또 출력

2.당첨 내역 출력

3.수익률 출력 
 
### 3. `LottoShop` 클래스
개수만큼 로또 출력

### 4. `LottoChecker` 클래스
당첨 확인 및 수익률 계산 

### 5. `LottoRank` ENUM 
enum 타입으로 로또 당첨 순위

### 6. `LottoService` 클래스
로또 기능을 실행
