# java-lotto-precourse

# 구현할 기능 목록
- 로또 구입 금액, 번호를 입력 받는 기능
  - 구입 금액은 천원 단위로 입력받을 수 있습니다.
    - 만약 8,450원을 입력하면 `IllegalArgumentException` 을 발생시키고 에러 문구를 출력합니다.
  - 로또 번호는 1 ~ 45 사이의 값만 받을 수 있습니다.
    - 그 외의 값을 입력받으면 `IllegalArgumentException` 을 발생시키고 에러 문구를 출력합니다.
- 로또 번호를 생성 하는 기능
- 입력한 로또 번호와 보너스 번호가 로또 번호와 일치하는지 확인하는 기능
- 로또의 수익률을 계산하는 기능
  - 소수점 둘쨰자리에서 반올림합니다.
- 출력 기능
  - 몇 개의 로또를 구입했는지, 해당 로또 번호가 어떻게 되는지(오름차순) 출력하는 기능 
  - 당첨 통계를 출력하는 기능
