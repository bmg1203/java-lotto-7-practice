# java-lotto-precourse

기능 요구사항
- 간단한 로또 발매기를 구현
- 1개의 로또 당 6개의 중복되지 않은 숫자를 뽑는다.
- 당첨 번호 추첨 시 중복되지 않은 숫자 6개와 보너스 번호 1개를 뽑는다.
- 당첨 기준과 금액
  - 1등: 6개 번호 일치 / 2,000,000,000원
    2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
  - 번호가 5개 일치했을 때만 보너스 번호 고려한다.  
    3등: 5개 번호 일치 / 1,500,000원
    4등: 4개 번호 일치 / 50,000원
    5등: 3개 번호 일치 / 5,000원
- 구입 금액 입력 시 구입 금액에 해당하는 만큼 로또 발행해야 한다.
- 당첨 번호와 보너스 번호를 입력받는다.
- 구매한 로또 번호와 당첨 번호를 비교, 당첨 내역 및 수익률을 출력하고 프로그램 종료한다.


입출력 요구사항
입력
- 잘못된 값을 입력했을 경우, IllegalArgumentException 발생시킨다.
  - "[ERROR]"로 시작하는 에러 메세지를 출력 후 그 부분부터 다시 입력받는다. 
  - 예시 : [ERROR] 로또 번호는 1부터 45 사이의 숫자여야 합니다.
- 1000원으로 나누어 떨어지는 로또 구입 금액으로 발행한 로또 수량 및 번호를 출력.
  - 1000원으로 나누어 떨어지지 않는 경우는 예외 처리
- 당첨 번호를 입력한다. 쉼표를 기준으로 구분한다.
- 그 후 보너스 번호를 입력한다.
출력
- 발행한 로또 수량 및 번호를 출력한다. 로또 번호는 오름차순으로 정렬해서 출력한다.
- 당첨 내역을 출력한다.
  - 3개 일치 (5,000원) - 1개
    4개 일치 (50,000원) - 0개
    5개 일치 (1,500,000원) - 0개
    5개 일치, 보너스 볼 일치 (30,000,000원) - 0개
    6개 일치 (2,000,000,000원) - 0개
- 수익률은 소수점 둘째 자리에서 반올림한다.
  - ex. 62.12% -> 62.1%
