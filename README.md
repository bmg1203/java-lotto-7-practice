| 핵심 개념  |
|:------:|
| **로또** |

# 기능 목록
## Presentation
- [ ] 구입금액을 입력 받는다
- [ ] 발행한 로또 수량과 번호를 출력한다
- [ ] 당첨 번호를 입력 받는다
- [ ] 보너스 번호를 입력 받는다
- [ ] 당첨 통계를 출력한다

## Service & Domain
- [ ] 로또를 발행한다
- [ ] 당첨 번호를 생성한다
- [ ] 당첨 내역을 계산한다
- [ ] 수익률을 계산한다
