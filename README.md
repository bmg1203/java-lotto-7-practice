# java-lotto-precourse


# 기능 요구 사항

## 핵심 기능
- [] 로또 번호를 랜덤으로 생성한다.
- [] 로또 번호를 저장한다. (오름차순 정렬)
- [] 로또 당첨 통계 로직을 구현한다.
  - [] 로또번호와 당첨번호를 비교하고 당첨 개수를 계산
  - [] 당첨 번호 개수가 5인 것 중, 보너스 번호와 일치하는지 확인
  - [] 수익률을 계산하는 기능


## 입력 기능
- [] 구입 금액을 입력받는다.
- [] 당첨 번호를 입력받는다.
- [] 보너스 번호를 입력받는다.


## 출력 기능
- [] 구입 금액에 따라 로또를 출력한다.
- [] 당첨 내역을 출력한다.
- [] 수익률을 출력한다.
