## 로또

### 📝 요구사항
- [x]  각각의 객체들을 싱글톤으로 관리하는 객체 생성
  → 스프링 컨테이너와 유사한 역할
- [x]  로또 구입 금액을 입력받는 객체 생성
  → Validation 진행
- [ ]  로또 구입 금액에 따라 로또 티켓을 생성
  → 로또 티켓은 여러 장 생성 가능
- [ ]  로또 당첨 번호를 생성하는 객체 생성
- [ ]  로또 당첨 번호와 로또 티켓을 비교
- [ ]  로또 당첨 통계를 출력
  → 3개 일치, 4개 일치, 5개 일치, 6개 일치
- [ ]  총 수익률을 계산
  → 로또 구입 금액 대비 당첨 금액의 비율
