# 로또

### 체크포인트

- [ ] 로또 숫자 범위 상수화
- [ ] 번호 발행, 중복되지 않는 숫자 6개
- [ ] 당첨 금액 상수화
- [ ] 입력
  - [ ] 로또 구입 금액에 따른 로또 발행
  - [ ] 로또 1장 가격 상수화
  - [ ] 당첨 번호 입력
  - [ ] 보너스 번호 입력
- [ ] 출력
  - [ ] 당첨 내역, 수익률 출력
- [ ] 예외 처리
  - [ ] 잘못된 값 입력
- [ ] 테스트
  - [ ] 각 테스트별 메서드 테스트
  - [ ] 실패 태스트
  - [ ] 성공 테스트
