## 1️⃣ _ 구현해야할 기능 리스트
```

    입력
        - 사용자에게 로또 구입 금액을 입력받는다.
        - 사용자에게 로또 당첨 번호를 입력받는다.
        - 사용자에게 로또 보너스 번호를 입력받는다.
    출력
        - 사용자가 입력한 로또 금액에 따라 구입한 로또의 결과를 출력한다.
        - 최종 당첨 통계와 최종 수익률을 출력한다.

    로직
        - 1-45 사이의 숫자를 6개 랜덤으로 뽑아 정렬한 숫자 배열을 만든다.
        - 랜덤한 숫자 배열을 만드는 행위를 N회 반복하며 로또 번호 배열을 업데이트한다.
        - 입력한 로또 번호를 기반으로 N등인지 판단하고.
        - 로또의 당첨 확인을 N회 반복하며 당첨 통계를 업데이트 한다.
        - 최종 수익률을 계산한다.
    예외
        - 로또 구매 금액이 숫자(Literal)인지 확인
        - 로또 구매 금액이 1,000원 단위로 나뉘어 떨어지는지
        - 당첨 번호 각각이 1~45 사이의 숫자인지 확인
        - 당첨 번호의 개수가 6개인지 확인
        - 보너스 번호가 1-45 사이의 숫자인지 확인
    DTO
        LottoDTO
            - 생성자 : 구입 금액을 입력받아 객체 인스턴스 생성
            - 새로운 로또 번호를 업데이트(추가)하는 함수
        WinnerDTO
            - 생성자 : 당첨 번호를 입력받아서 객체 인스턴스 생성 
            - 당첨 결과를 업데이트 하는 함수
    Enum
        - 로또 등수에 따른 당첨 금액을 저장

```
