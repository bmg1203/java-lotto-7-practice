# java-lotto-precourse

## 구현할 기능 목록

1. **구매 금액 입력**
    - 로또 구입 금액을 입력받고, 1,000원 단위로 검증.

2. **로또 발행**
    - 입력된 금액에 따라 로또 발행.
    - 로또 번호는 1~45 중 중복되지 않는 6개 숫자.

3. **당첨 번호 입력**
    - 당첨 번호 6개와 보너스 번호 1개를 입력받음.
    - 번호는 1~45 범위 내 중복되지 않음.

4. **당첨 결과 계산**
    - 구매한 로또와 당첨 번호 비교.
    - 1등부터 5등까지 당첨 결과 도출.

5. **당첨 통계 출력**
    - 각 등수별 당첨 개수와 금액 출력.
    - 총 수익률 계산 및 출력.

6. **예외 처리**
    - 잘못된 입력 시 `IllegalArgumentException` 발생.
    - "[ERROR]"로 시작하는 에러 메시지 출력 후 재입력 요구.
