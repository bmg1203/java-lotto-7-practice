- [x] 당첨 내역을 계산하는 기능

## 테스트
1. view - InputView
   - 사용자 입력 검증
     - 지불액이 1000단위가 아닐 경우
     - 지불액이 음수인경우
     - 당첨 번호 형식
     - 당첨 번호 6개 초과
     - 당첨 번호 6개 미달
     - 당첨 번호 범위
     - 보너스 번호 형식
     - 보너스 번호 범위
     - 보너스 번호 당첨 번호와 중복
2. domain - Lotto
   - 로또 번호가 중복되는 경우
   - 로또 번호 6개 초과
   - 로또 번호 6개 미달
   
3. model - LottoModel
   - 로또 번호 정렬
   - 당첨 등수 집계
   - 수익률 계산
   
3. controller - LottoController
    - 로또 일반 기능 테스트
    - 당첨 테스트
    - 비당첨 테스트
