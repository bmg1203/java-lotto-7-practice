# java-lotto-precourse

### 🦖 기능 목록

**1. 입력 기능**

    1-1) 구입 금액 입력 (정수 또는 1000원 단위가 아니거나 음수라면 예외처리)
    1-2) 당첨 번호 입력 (정수 또는 로또번호 범위가 아니거나 음수라면 예외처리)
    1-3) 보너스 번호 입력 (정수 또는 로또번호 범위가 아니거나 음수라면 예외처리)

**2. 로또 기능**

    2-1) 구입 금액에 맞게 로또 발행
    2-2) 당첨 여부 확인
    2-3) 수익률 계산

**3. 출력 기능**
    
    3-1) 발행 로또 수량 및 번호 출력
    3-2) 당첨 내역 출력
    3-3) 수익률 출력
