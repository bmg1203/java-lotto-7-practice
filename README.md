## 제약 조건
1. 구매액은 반드시 1,000원 단위의 int 범위 이내의 자연수
2. 당첨 번호는 쉼표로 구분되는 1부터 45 사이의 중복되지 않는 자연수 6개
3. 보너스 번호는 1부터 45 사이의 자연수 1개

## 기능 구현 목록
1. 로또 자동 번호 구매
2. 당첨 번호 입력
3. 로또 당첨 로직
4. 통계 계산, 출력

## 입출력
> 입력 :

1. 천원 단위의 로또 구매액
2. 당첨 번호 6개
3. 보너스 번호 1개

> 예시
```
11000
1,2,3,4,5,6
7
```

> 출력 :

1. 로또 자동 번호(오름차순)
2. 당첨 통계 

> 예시
```
11개를 구매했습니다 ~~
```
```
당첨 통계
```
