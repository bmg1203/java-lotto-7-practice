# java-lotto-precourse

# 구현할 기능 목록
    1. Lotto 클래스
        - 번호 입력 및 검증
        - 로또 번호 리스트 반환
    2. Selling 클래스
        - 가격 검증
        - 구입 금액에 따른 랜덤 번호 리스트 반환
        
    3. Winning 클래스
        - 입력한 로또 번호와 당첨 번호를 비교
        - 각 로또의 당첨 등수를 기록
        - 당첨 결과를 출력
        - 보너스 번호 관련 처리
        - 중복된 숫자 개수 계산

    4. Benefit 클래스
        - 수익률 계산
        - 수익률 출력
