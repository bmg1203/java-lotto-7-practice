# 구현할 기능 목록
- [ ] 구입 금액 입력
- [ ] 중복 되지 않는 숫자 6개 반환
- [ ] 숫자 오름차순으로 정렬
- [ ] 발행한 로또 수량 및 번호 출력
- [ ] 

# 예외 상황 목록

- [ ] *로또 구입 금액 예외* 
    - [ ] 로또 구입 금액이 숫자가 아닌 경우
    - [ ] 로또 구입 금액이 1000원 미만인 경우
- [ ] *로또 당첨 번호 예외*
  - [ ] 로또 당첨 번호가 숫자가 아닌 경우
  - [ ] 로또 당첨 번호가 1에서 45 사이의 수가 아닌 경우
  - [ ] 보너스 번호가 숫자가 아닌 경우
  - [ ] 보너스 번호가 1에서 45 사이의 수가 아닌 경우
