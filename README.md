# java-lotto-precourse

## 기능 요구사항

#### 입력
- [x] 사용자는 로또 구입 금액을 입력할 수 있다.
- [x] 사용자는 당첨 번호를 입력한다. 
- [x] 로또는 중복된 숫자를 가질 수 없다. (Validation)
- [x] 사용자는 보너스 번호를 입력한다. 


#### 내부 기능 
- [x] 로또는 1부터 45까지의 숫자 중 6개를 랜덤으로 뽑아 로또를 구성한다.

- [x] 로또별 당첨 여부 계산
- [x] 당첨 통계 계산
- [x] 수익률 계산


#### 출력
- [x] 사용자의 구입 금액을 기반으로 사용자가 구매한 로또 갯수를 출력한다.
- [x] 발행한 로또 번호를 수량에 맞추어 출력한다.
- [x] 당첨 통계를 출력한다. 
- [x] 수익률을 출력한다.
